사용 가이드

DrugMaster Validator 사용법 및 결과 해석 가이드

01

DB 파일 등록

먼저 DB 파일 등록 메뉴에서 약가마스터_성분중심_보완.sqlite 파일을 업로드합니다. 파일은 브라우저에 자동 저장되어 다음에 자동 복원됩니다.

모든 처리는 브라우저 내에서만 진행됩니다 (서버 전송 없음)
02

파일 준비

검증할 약제 목록을 Excel(.xlsx) 또는 CSV 형식으로 준비합니다.

필수 컬럼
주성분코드성분명투여분류
03

판정 결과 해석

  • 최신 유효 현재 급여 등재 확인됨
  • 참고사항 코드 일치, 성분명/투여 차이
  • 과거이력 최신에 없지만 과거 존재
  • 미확인오류 코드/이력 모두 불명확
04

누락리스트

같은 성분군 안에서 마스터에는 있지만 제출 목록에 없는 코드를 자동 추출합니다.

  • 상 필수 – 동일 투여경로 내 누락
  • 중 확장 – 동일 제형군 내 누락
  • 하 참고 – 관련 제형군 내 누락
05

신뢰도 기준

  • 높음 코드변경이력 또는 완전 일치
  • 중간 성분명·투여 일치 추정
  • 낮음 앞 4자리 주성분군만 일치
"낮음"은 반드시 수동 확인 필요
06

결과 다운로드

검증 완료 후 결과 추출 도구 페이지에서 전체 결과, 미확인오류, 누락상세, 과거이력 검토표를 각각 Excel 파일로 저장할 수 있습니다.

실제 .xlsx 파일이 로컬에 저장됩니다
