## [2.0.0] - 2026-04-10

### 추가됨 (Added)
- **동일주성분별 업체수 분석 기능**
  - 주성분코드 앞 4자리 기준으로 성분군 그룹핑
  - 성분군별 업체수, 제품수 자동 집계
  - 시장집중도 자동 판정 (독점/과점/경쟁/치열)
  
- **새로운 결과 시트: "동일주성분별_업체수"**
  - 주성분군(앞4자리), 대표 주성분명, 총 제품수
  - 업체수, 업체목록, 투여경로
  - 시장집중도, 비고
  - 업체수 기준 색상 구분 (빨강/주황/연두/회색)
  
- **행별검증 시트 신규 컬럼**
  - `동일주성분_업체수`: 해당 코드가 속한 성분군의 업체수
  - `시장집중도`: 독점/과점/경쟁/치열 자동 표시
  
- **요약 시트 신규 통계**
  - 전체 성분군 수
  - 평균 업체수
  - 최대 업체수 성분군
  - 독점/과점/경쟁/치열 성분군 각각의 개수
  
- **새로운 별도 추출 기능**
  - 독점/과점 성분군 별도 파일 생성
  - 시장 집중도 높은 품목 우선 검토용

### 변경됨 (Changed)
- 검증 로직에 8단계 추가: "동일주성분별 업체수 분석"
- 사용자 응답 형식에 업체수 인사이트 추가
- 색상 규칙 확장 (시장 경쟁도 반영)
- 기본 운영 철학에 "시장 경쟁 구조 분석" 추가

### 기술적 세부사항
- **그룹핑 기준**: 주성분코드 앞 4자리
  - 9자리 전체 사용 시 과도한 세분화 방지
  - 주성분의 화학적 계열/분류 기준으로 시장 구조 파악
  
- **업체수 계산**: 업체명 기준 중복 제거
  - 동일 업체의 여러 제품 = 1개 업체로 계산
  
- **시장집중도 판정 기준**:
  - 독점: 업체수 1개
  - 과점: 업체수 2~4개
  - 경쟁: 업체수 5~9개
  - 치열: 업체수 10개 이상

### 활용 사례
1. **신규 진입 검토**: 성분군별 경쟁 강도 파악
2. **가격 협상 전략**: 독점/과점 품목 식별
3. **누락 우선순위**: 독점 품목 누락 시 우선 검토
4. **시장 분석**: 성분군별 시장 집중도 리포트

## 업그레이드 가이드

### v1.x → v2.0 업그레이드

**호환성:**
- ✅ 기존 DB 파일 그대로 사용 가능
- ✅ 기존 검증 로직 모두 유지
- ✅ 추가 기능이므로 이전 결과 파일과 충돌 없음

**새로운 기능 사용법:**
1. 약제 파일 검증 시 자동으로 업체수 분석 수행
2. 결과 엑셀 파일에서 "동일주성분별_업체수" 시트 확인
3. 행별검증 시트에서 각 코드의 시장집중도 즉시 확인
4. 요약 시트에서 전체 시장 구조 통계 파악

**추가 작업 불필요:**
- DB 재구축 불필요
- 설정 변경 불필요
- 자동으로 모든 검증에 적용됨
